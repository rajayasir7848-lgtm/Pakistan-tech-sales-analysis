Pakistan Tech Sales Analysis

Overview:

This is my first data analysis project where I analyzed tech product sales across major Pakistani cities using Python and SQL

Tools Used:
Python
Pandas
Matplotlib
SQL (SQLite)
Excel (openpyxl)

Key Findings:

➟Karachi is the top revenue city with 4,430 total revenue
➟Laptop generates 70% of total revenue despite selling only 12 units
➟Bilal is the top salesperson with 4,425 revenue
➟Peak revenue day was January 15 with 3,150 in a single day
➟Mouse is best selling by quantity (49 units) but lowest revenue contributor
➟Electronics category outperforms Accessories by 4.6x

Project Structure:

data/ — raw and cleaned CSV files
charts/ — all visualizations
sql_results/ — SQL query screenshots
sales.ipynb — main analysis notebook
pakistan_sales_report.xlsx — Excel report
